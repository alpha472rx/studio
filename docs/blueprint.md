# **App Name**: Vanguard Assistant

## Core Features:

- Aim Assistant Toggle: Enable or disable aim assistance to improve targeting during gameplay.
- Anti-Cheat Bypass: Toggle to attempt bypassing anti-cheat detection mechanisms. THIS FEATURE IS FOR EDUCATIONAL PURPOSES ONLY. DO NOT USE IN REAL GAME.
- Aimbot Target Selection: Select the body part (head, body, neck) for the aimbot to target.
- Aimlock Toggle: Toggle aimlock to lock the aim on the selected target.
- Adjustable Aim FOV: Set the field of view for the aimbot, controlling how wide the aim assist is applied.
- ESP Toggle: Toggle Extra Sensory Perception (ESP) to display information about other players and entities.

## Style Guidelines:

- Primary color: Deep dark blue (#0F3460) provides a strong base.
- Background color: Very dark blue (#1A1A2E), nearly black, matches the overall dark theme.
- Accent color: Vivid pink (#E94560) creates highlights.
- Font: 'Segoe UI', sans-serif; the user requests this font and others cannot be used. Note: currently only Google Fonts are supported.
- Mobile-first design ensures responsiveness and adaptability to various screen sizes.
- Simple toggle icons
